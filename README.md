# Пазл с таблицей умножения

Статическая HTML/CSS/JS-игра для GitHub Pages.

## Как заменить картинку

Положите файл `pic.jpg` рядом с `index.html`. Игра автоматически возьмет эту картинку для пазла.

Лучше всего подходит изображение с пропорцией около `16:10`, например `1600x1000`.

## Публикация на GitHub Pages

1. Загрузите в репозиторий файлы `index.html`, `styles.css`, `script.js`, `.nojekyll`, `README.md` и свою картинку `pic.jpg`.
2. В GitHub откройте `Settings` -> `Pages`.
3. В `Build and deployment` выберите `Deploy from a branch`.
4. Выберите ветку `main` и папку `/root`.
5. Нажмите `Save`.

После публикации сайт будет доступен по ссылке GitHub Pages для репозитория.
